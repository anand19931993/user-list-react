The assignment is done using json-server as backend for fetching dynamic values and also updating them.
1. Run - npx json-server --watch src/db.json in one terminal.
   This will run the mock api for the app. By default this will run in port 3000.
2. Open another terminal. Unzip the zip in a folder. Use npm install to install all dependencies.
3. npm start and it will ask you to run in another port say "y" and app will start.
